# Genius Academy - Design Style Guide

## Brand Identity

### Brand Name
**GENIUS ACADEMY** - Where Learning Never Stops

### Color Palette (Strict - Only These 5 Colors)
- **Royal Blue** (#2563EB) - Primary color, main branding
- **Bright Yellow** (#FCD34D) - Accent, highlights, CTAs
- **Strong Red** (#DC2626) - Alerts, warnings, important actions
- **Black** (#000000) - Headings, strong text, navigation
- **White** (#FFFFFF) - Backgrounds, cards, content areas

### Typography System
- **Raleway** - Logo and big titles ("GENIUS ACADEMY", "Student Dashboard")
- **Poppins** - Section headings and card titles
- **Source Sans 3** - Body text, descriptions, paragraphs
- **Montserrat** - All buttons, navigation items, labels, CTAs

## Visual Language

### Design Principles
- Modern, clean, dashboard-like interface
- Rounded cards with subtle shadows
- Clear spacing and visual hierarchy
- Consistent use of brand colors
- Professional educational aesthetic

### Component Styling
- **Cards**: White background, rounded corners (8px), subtle shadow
- **Buttons**: Montserrat font, rounded (6px), consistent padding
- **Forms**: Clean inputs with focus states in brand colors
- **Navigation**: Clear hierarchy with active states
- **Modals**: Centered with backdrop blur effect

## Visual Effects

### Used Libraries
- **Anime.js** - Smooth animations and transitions
- **ECharts.js** - Interactive data visualization
- **Splide.js** - Image carousels and sliders
- **p5.js** - Creative background effects
- **Pixi.js** - Advanced visual effects

### Animation Effects
- **Text Effects**: 
  - Color cycling emphasis on key headings
  - Split-by-letter stagger animations
  - Gradient text animation for brand name
- **Image Effects**:
  - Ken Burns pan/zoom on hero images
  - Infinite image scroller for testimonials
  - Hover animations on course cards
- **Background Effects**:
  - Subtle particle system for hero section
  - Aurora gradient flow for dashboard backgrounds
  - Liquid-metal displacement effects

### Header Effect
- Animated gradient background using CSS and JavaScript
- Floating particles for dynamic visual interest
- Smooth color transitions matching brand palette

### Scroll Motion
- Subtle reveal animations for content sections
- Parallax effects on hero elements
- Smooth transitions between sections
- Progressive disclosure of dashboard features

### Hover Effects
- **Cards**: Subtle lift with shadow expansion
- **Buttons**: Color transition and slight scale
- **Images**: Zoom with overlay information
- **Navigation**: Underline animation and color change

## Layout System

### Grid Structure
- 12-column responsive grid system
- Consistent spacing using 8px base unit
- Mobile-first responsive breakpoints
- Flexible card layouts for content

### Spacing Guidelines
- **Sections**: 64px vertical spacing
- **Cards**: 24px internal padding
- **Elements**: 16px standard spacing
- **Text**: 8px line height adjustments

### Responsive Design
- **Desktop**: Full dashboard layout with sidebar
- **Tablet**: Collapsible navigation with main content
- **Mobile**: Stack layout with bottom navigation

## Interactive Elements

### Button Styles
- **Primary**: Royal blue background, white text
- **Secondary**: Yellow background, black text
- **Danger**: Red background, white text
- **Ghost**: Transparent with colored border

### Form Elements
- **Inputs**: Clean borders with blue focus states
- **Dropdowns**: Custom styled with brand colors
- **Checkboxes**: Custom design with smooth transitions
- **File Uploads**: Drag-and-drop with visual feedback

### Navigation States
- **Active**: Blue background with white text
- **Hover**: Light blue background
- **Disabled**: Grayed out with reduced opacity
- **Mobile**: Hamburger menu with slide-out drawer
